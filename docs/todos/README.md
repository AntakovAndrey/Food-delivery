## Как использовать

1. Начните с первого сервиса (Identity Service) - он является основой для всех остальных
2. Отмечайте выполненные задачи, добавляя `[x]` вместо `[ ]`
3. Следуйте приоритетам, указанным в каждом файле
4. Обновляйте статус задач по мере выполнения
